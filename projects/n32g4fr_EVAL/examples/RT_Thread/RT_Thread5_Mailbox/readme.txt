2. 使用环境
    硬件环境：工程对应的开发硬件平台 
    开发板：   N32G4FRH_STB V1.1

    描述相关模块配置方法；例如:时钟，I/O等 
         1. SystemClock：144MHz
         2. GPIO：KEY1--PA4, KEY2--PA5, KEY3--PA6
                        日志：TX--PA9  RX--PA10，波特率：115200

    描述Demo的测试步骤和现象 
         1. 编译后下载程序复位运行
         2. 接好串口打印工具
         3. 本例程创建两个线程，key 线程和 test 线程，key 线程检测按键发送邮件，test 线程接收邮件，按下按键（KEY1、KEY2、KEY3）发送邮件，查看打印日志
